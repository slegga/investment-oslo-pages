# Investment Oslo Pages

Offentlig GitHub Pages for investeringsporteføljen.

## Beskrivelse

Dashbord som viser portefølje-data fra investment-oslo.

## Data

- `data/portefolje.json` - Portefølgedefinisjoner
- `data/posisjon.json` - Aksjeposisjoner
- `data/portefoljeutvikling.json` - Historisk utvikling

## Oppdatering

Kjør `publish.sh` fra investment-oslo for å oppdatere data og Publich Pages.
